# vulncentral-db

Paquete compartido: `Base` declarativo, mixins, modelos ORM y hashing bcrypt usados por **api-gateway** y **worker** contra la misma PostgreSQL.

Instalación local (desde la raíz del repo):

```bash
pip install -e packages/vulncentral-db
```

Las migraciones Alembic siguen en `services/api-gateway/alembic`.
